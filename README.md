RAGVerifier

A framework for analyzing retrieval and faithfulness failures in Retrieval-Augmented Generation (RAG) systems.



Motivation

RAG systems improve factual grounding by retrieving external documents before generating answers. 
However, when answers are incorrect, it is often difficult to determine whether the failure originated from retrieval, insufficient evidence, or unsupported generation, basically unclear.

This RAGVerifier aims to understanding and diagnosing these failures and provide clarity.



Project Goals

- Evaluate retrieval quality
- Analyze evidence support
- Investigate answer faithfulness
- Categorize common failure modes
- Improve transparency of RAG evaluation
